📋 AWS Setup Checklist

Do this once in the AWS Console before using the app:

  1. Create an S3 bucket called prerana-farm-photos (or any name) in ap-south-1.
  2. In the bucket → Permissions → Block Public Access → uncheck "Block all public access" → Save.
  3. Add a Bucket Policy to allow public reads (so photos display in the gallery):

    {"Version":"2012-10-17","Statement":[{"Effect":"Allow","Principal":"*","Action":"s3:GetObject","Resource":"arn:aws:s3:::YOUR-BUCKET/*"}]}
  4. Add a CORS policy to allow uploads from this page:

    [{"AllowedOrigins":["*"],"AllowedMethods":["GET","PUT","POST"],"AllowedHeaders":["*"],"ExposeHeaders":["ETag"]}]
  5. Create an IAM user with AmazonS3FullAccess → generate Access Key → paste above.

Tip: The metadata file tags/index.json is auto-created on first upload and updated on each new tag. The gallery reads from it to show all photos — no database needed.
